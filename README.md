# pyDentate

This repository will be dedicated to the reproducibility of pyDentate, a computational model used in a paper currently under submission. We will update the information here
after final decision. For the newer version of pyDentate visit the active repository [here](https://github.com/danielmk/pyDentate). If you have any questions or suggestions
regarding pyDentate reproducibility you can directly contact danielmuellermsc@gmail.com.
